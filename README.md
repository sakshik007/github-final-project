#  Simple Interest Calculator

A lightweight, open-source tool designed to quickly compute simple interest. 

##  How It Works

The application calculates interest using the standard financial formula:

$$A = \frac{P \times T \times R}{100}$$

Where:
* **P** = Principal Amount (the initial sum of money)
* **T** = Time (in years)
* **R** = Rate of Interest (annual percentage)

##  Features

* **Instant Calculation:** Get accurate simple interest results in seconds.
* **Minimalist Design:** Clean, user-friendly, and straightforward interface.
* **Open Source:** Free to use, modify, and distribute under the MIT License.
